# Minecraft 2D PYTHON – Desenvolvimento de um jogo estilo sandbox utilizando Pygame e Programação Orientada a Objetos.

Link do Curso: https://www.udemy.com/share/109TQc3@29DB3fm2OFNPo6_z6zr2WIDBmuMzhgvNDrpHYKqUTWrJGDKgZpPO5_-FDCgJMuWyMA==/

Descrição: Este repositório contém o código desenvolvido durante o curso do instrutor Renato Ikeuchi. O projeto consiste na criação de um Minecraft em 2D, onde foram aplicados conceitos fundamentais de Game Design e lógica de programação. O software resolve desafios de renderização de mundo, otimização de câmera, persistência de dados (save) e gerenciamento de inventário, tudo estruturado sob o paradigma de Programação Orientada a Objetos (POO).

Pré-requisitos:

    Python 3.x instalado.

    IDE PyCharm (recomendado).

    Biblioteca Pygame.

Instalação/Uso:

    Clone este repositório para sua máquina local.

    Abra a pasta do projeto no PyCharm.

    Certifique-se de que a biblioteca Pygame está instalada no seu ambiente virtual. Caso não esteja, instale via terminal com: pip install pygame

    Localize o arquivo principal (ex: main.py) e execute-o.

    Comandos: Utilize o teclado e o mouse para interagir com o mundo, construir estruturas, gerenciar a barra de itens e enfrentar inimigos.

Tecnologias:

    Python

    Pygame

    PyCharm

Funcionalidades Desenvolvidas:

    Mundo e Renderização: Construção de cenários, sistema de câmera dinâmica e otimizações de performance.

    Mecânicas de Player: Sistema de vida, barra de itens (HUD), bolsa (bag) e sistema de inventário funcional.

    Entidades: Adição de animais e inteligência básica para inimigos.

    Persistência: Sistema para salvar e carregar dados do progresso do jogador.

    POO: Estruturação de classes para objetos, entidades e gerenciamento do estado do jogo.

Contribuição:
Este projeto foi concluído como parte de um marco de aprendizado em Python. Feedbacks sobre a estrutura das classes ou propostas de novas melhorias no motor do jogo são bem-vindos através de pull requests.
